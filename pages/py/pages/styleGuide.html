Paradise Yogurt style guide

Palette

The colors we chose were for use as borders and texts. We also used some in the logo and for backgrounds

Use following colors

Orange #de562c: Can Be specified using class="orangeBack" for background or class="orangeText"
Yello #f4e53b: Can Be specified using class="yellowBack" for background or class="yellowText" for text
Light Purple #b784ef: Can Be specified using class lightPurpleBack for background or lightPurpleText for text
Dark Purple #ad14bf: Can Be specified using class darkPurplBack for background or darkPurpleText for text
Blue #1371df: Can Be specified using class blueBack for background or blueText for text

Typography

  • H1 Main Headlines Solway

    Solway, ExtraBold, or Lato, 2.25em

  • H2 Sub Headlines Solway

    Solway, Bold, 1.5em

  • H1 Lato

    Lato, 2.25em

  • H2 Lato

    1.50em

  • H3 Lato

    1em

  • Paragraph text. Lato, Regular, 1em

  • All h1 tags are set to 2.25ems and can be either solway or lato. To use either you just need to add a class="solway" or class="lato" as defined in the css file.
